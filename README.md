1. 용기(Courage) : 고객의 요구사항 변화에 능동적인 대처
2. 단순성(Simplicity) : 부가적 기능, 사용되지 않는 구조와 알고리즘 배제

4. 피드백(FeedBack) : 지속적인 테스트와 반복적인 결함 수정, 빠른 피드백
